A simple Script for compression Video and Frame Sequences to mp4 x264/aac. 
In most situations the settings are ok, but you can also edit it in the batch file. 
The best place for the Script is the SendTo menu from Windows. 
Frame Sequences must have minimum 4 Numbers at the end: *0000.ext

The media_compressor_v0.9.bat support also Audio File compression - mp3, wav, flac or what you want.
